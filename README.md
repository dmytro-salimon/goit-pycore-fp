# 🤖 Personal Assistant Bot — Твій інтелектуальний CLI-помічник

Цей застосунок є повноцінною системою для керування контактами та нотатками через зручний інтерфейс командного рядка (CLI). Він дозволяє не лише зберігати важливі дані, а й швидко знаходити їх, редагувати та отримувати нагадування про важливі події.

## 🛠️ Встановлення та налаштування

Проєкт можна встановити як локальний пакет, що дозволяє запускати його з будь-якого місця системи.

1. **Клонуйте репозиторій та перейдіть у робочу директорію.**
2. **Створіть та активуйте віртуальне середовище:**

```bash 
python3 -m venv .venv
source .venv/bin/activate
```

3. **Встановіть проєкт:**

```bash 
pip install.
```

## 🚀 Як користуватися
Після встановлення ви можете запустити програму з будь-якої точки термінала за допомогою команди:

```bash 
assistant-bot
```

Програма працює в інтерактивному режимі: вона чекає на ваші команди, виконує їх і знову пропонує введення, поки ви не вирішите вийти.

**Початок роботи**
- Введіть `help`, щоб у будь-який момент переглянути список усіх доступних команд та їх формат.
- Введіть `hello`, щоб перевірити готовність бота до роботи.
- Введіть `close` або `exit`, щоб надійно зберегти всі зміни та вийти.

## 📋 Довідник команд

**👤 Робота з контактами**

Ви можете зберігати імена, телефони, email, адреси та дні народження.

| Команда | Формат | Опис |
| :--- | :--- | :--- |
| **Додати** | `add <Name> <Phone>` | Створює новий контакт або додає телефон до існуючого |
| **Змінити** | `change <Name> <OldPhone> <NewPhone>` | Замінює старий номер на новий для вказаного контакту. |
| **Телефони** | `phone <Name>` | Показує всі номери телефону конкретної людини. |
| **Показати всі** | `all` | Виводить повний список контактів у зручному форматі. |
| **День народження** | `add-birthday <Name> <DD.MM.YYYY>` | Додає дату народження контакту (з валідацією). |
| **Коли свято** | `show-birthday <Name>` | Показує збережену дату народження. |
| **Іменинники** | `birthdays [days]` | Список контактів, чий день народження настане через X днів. |
| **Пошук** | `search <Query>` | Пошук контактів за частиною імені або номера телефону. |
| **Видалити** | `delete <Name>` | Повне видалення контакту з бази. |

**🗒️ Управління нотатками**

Нотатки підтримують текстовий зміст та систему хештегів для зручного групування.

| Команда | Формат | Опис |
| :--- | :--- | :--- |
| **Створити** | `add-note <Text> [#tags]` | Створює нотатку. Теги вказуються через символ `#`. |
| **Список** | `notes` | Виводить всі нотатки з їх унікальними ID. |
| **Пошук** | `search-note <Text>` | Знаходить нотатки, що містять вказане слово. |
| **За тегом** | `search-tag <Tag>` | Знаходить всі нотатки за вказаним тегом. |
| **Редагувати** | `edit-note <ID> <NewText>` | Змінює зміст нотатки за її ідентифікатором. |
| **Видалити** | `delete-note <ID>` | Видаляє нотатку за її ідентифікатором. |

## 💡 Приклади використання (Кейси)

1. **Додавання контакту з валідацією:**
- `add Dmytro 0991234567` — успішно.
- `add Dmytro 123` — бот видасть помилку: "Номер телефону має складатися рівно з 10 цифр".

2. **Робота з іменинниками:**
- add-birthday Dmytro 10.04.1995.`
- `birthdays 14` — виведе всіх, у кого день народження протягом наступних двох тижнів.

3. **Створення нотатки з тегами:**
- `add-note Купити молоко та хліб #покупки #дім.`
- `search-tag покупки` — бот знайде цю нотатку за тегом.

## 💾 Збереження даних

Вам не потрібно хвилюватися про втрату даних. Бот автоматично зберігає Address Book та Notes Book на жорсткий диск при кожній зміні та перед виходом із програми. При наступному запуску всі дані будуть відновлені автоматично.
